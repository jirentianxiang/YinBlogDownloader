# yinwangdl
download Yin Wang all blog and convert to pdf.

## Dependency
* wkhtmltopdf
* python-pdfkit
* beautifulsoup4

``` shell
sudo apt install wkhtmltopdf
pip3 install pdfkit beautifulsoup4
```

Please check [python-pdfkit](https://github.com/JazzCore/python-pdfkit) for the details.
